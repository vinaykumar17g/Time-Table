Suggested Projects for Development

Build these projects to enhance your skills and resume.

Focus on understanding the underlying concepts, clean code, and version control (Git).

  • Beginner Android Projects (Phase 1):
    • To-Do List App: Basic CRUD functionality. Learn about `Activities`, `Intents`, and `SharedPreferences`.
    • Unit Converter: Simple UI with input fields and conversion logic. Focus on responsive design.
    • Weather App: Use a simple API to display weather data. This introduces network requests.
  • Intermediate Android Projects (Phase 3):
    • Notes App with Room DB: A more robust to-do or notes app that uses a local database (`Room`) and modern architecture (`MVVM`).
    • News Aggregator App: Fetches news articles from an API, displays them in a `RecyclerView`, and includes features like filtering or search.
    • Simple Chat App: Implements real-time messaging using Firebase.
